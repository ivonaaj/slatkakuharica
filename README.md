# slatkakuharica
Ova web stranica sadrži slatke recepte, podjeljene u 3 kategorije: kolači, torte i veganski. Također stranica je dizajnirana tako da je laka za snalaženje, pregledna je te korinsik ima mogućnost filtriranja rezultata kao i pretragu istih po nazivu slastice. Veliki plus je što u ovoj online kuharici možete napisati i poslati svoj recept.
