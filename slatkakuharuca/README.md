#  SLATKA KUHARICA

Slatka kuharica je web stranica sa slatkim receptima za kolače, torte i veganske deserte. Stranica omogućuje pregled recepata, pretraživanje i sortiranje recepata na 3 načina te preuzimanje recepata u PDF formatu. Korisnici također mogu poslati vlastiti recept putem forme za unos recepata i tako pomoći u napredovanju ove male slatke kuharice.

##  Live demo

https://slatkakuharica.netlify.app

##  Stranice

- index.html – Naslovnica
- kolaci.html – Recepti za kolače
- torte.html – Recepti za torte
- veganski.html – Veganski recepti
- kontakt.html – Forma za slanje vlastitih recepata

## Tehnologije

- HTML5
- CSS3 (Flexbox, Grid, CSS varijable)
- JavaScript (vanilla JS)
- Google Analytics 4
- Netlify hosting

##  Funkcionalnosti

- Pretraživanje recepata po nazivu
- Sortiranje recepata prema nazivu tj. abecedom od A do Z, vremenu pripreme i težini
- Preuzimanje recepata na svoj uređaj u PDF formatu
- Responsive dizajn prilagođen mobilnim uređajima i desktopu
- Forma za slanje recepata s JavaScript validacijom
- Google Analytics praćenje posjeta stranici
- Preglednost i lako snalaženje na stranici
